Recommendations that must be addressed in further submissions:

This enhancement will pave the way for more robust data handling and processing capabilities.
  • Sensor Calibration for Improved Accuracy: The VL53L0/1XV2 - TOF 400 IR Ranging sensor you've chosen is commendable. However, the calibration process needs refinement. Instead of averaging readings over a short span, a more meticulous approach involving manual serial array calibration for each millimeter is essential for precision. Please refer to the detailed guidelines provided in the ST datasheet (VL53L0/1XV2 - STs Data Sheet) for a comprehensive understanding of the calibration process.
  • Optical Component Optimization: To achieve enhanced accuracy, consider transitioning from using LDRs to Phototransistors or Photodiodes. In scenarios where LDRs are the only option, integrating a shield type LDR is advisable to mitigate external interferences.
  • Motor Stability: The current use of DPDT switches for motor control could be improved. While incorporating a motor driver is the optimal solution, if constraints prevent this, it's crucial to ensure the DPDT switch is meticulously calibrated for stable operation.
  • Control Panel Implementation: Developing a control panel will offer a more dynamic and user-friendly interface, reducing the reliance on hardcoded parameters and allowing for real-time adjustments and monitoring.
